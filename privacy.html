Privacy Policy

EngageBack for X

EngageBack for X is a local-first Chrome extension that helps users remember their own visible engagement activity on X.com and Twitter.com.

Effective Date

May 23, 2026

Overview

EngageBack for X helps users review engagement patterns based only on X.com and Twitter.com pages they personally open in their browser.

EngageBack does not send user data to the developer, a backend server, or third parties. Data is stored locally in the user's browser.

Data Processed

EngageBack may process visible website content and user activity from X.com and Twitter.com pages that the user personally opens.

This may include:

  • Visible handles and display names
  • Post or status links
  • Short visible post snippets
  • Notification text visible on the page
  • Visible engagement indicators
  • User actions such as likes, replies, reposts, bookmarks, and profile visits when detectable

Local Storage

EngageBack stores data locally in the user's browser using Chrome extension storage and IndexedDB.

This local data is used only to provide the extension's dashboard, engagement summaries, local badges, suggested actions, exports, and user preferences.

No Data Transmission

  • EngageBack does not send user data to the developer.
  • EngageBack does not send user data to a backend server.
  • EngageBack does not sell or share user data.
  • EngageBack does not use user data for advertising.
  • EngageBack does not use analytics.
  • EngageBack does not use the X API.
  • EngageBack does not use user data to determine creditworthiness or for lending purposes.

No Automation

EngageBack does not automate likes, replies, reposts, follows, direct messages, or navigation.

Users remain fully responsible for any actions they choose to take on X.com or Twitter.com.

User Control

  • Users can export their local EngageBack data as JSON or CSV from the extension.
  • Users can delete all local EngageBack data from the extension at any time.
  • Users can disable tracking and page badges through extension settings when available.

Data Scope

EngageBack only runs on X.com and Twitter.com pages covered by its Chrome extension permissions.

EngageBack does not read or collect data from other websites.

Changes

This privacy policy may be updated from time to time. Updates will be posted at this same URL.
