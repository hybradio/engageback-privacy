EngageBack for X Privacy Policy Effective Date: May 23, 2026 EngageBack for X is a local-first Chrome extension that helps users remember their own visible engagement activity on X.com and Twitter.com. Data Processed EngageBack may process visible website content and user activity from X.com and Twitter.com pages that the user personally opens. This may include visible handles, display names, post/status links, post snippets, notification text, visible engagement indicators, and user actions such as likes, replies, reposts, bookmarks, and profile visits. Local Storage EngageBack stores data locally in the user's browser using Chrome extension storage and IndexedDB. This data is used only to provide the extension's dashboard, engagement summaries, local badges, suggested actions, exports, and user preferences. No Data Transmission EngageBack does not send user data to the developer, to a backend server, or to third parties. EngageBack does not use the X API. EngageBack does not use analytics. EngageBack does not sell or share user data. EngageBack does not use user data for advertising. EngageBack does not use user data to determine creditworthiness or for lending purposes. No Automation EngageBack does not automate likes, replies, reposts, follows, direct messages, or navigation. Users remain fully responsible for any actions they choose to take on X.com or Twitter.com. User Control Users can export their local EngageBack data as JSON or CSV from the extension. Users can delete all local EngageBack data from the extension at any time. Data Scope EngageBack only runs on X.com and Twitter.com pages covered by its Chrome extension permissions. It does not read or collect data from other websites. Changes This privacy policy may be updated from time to time. Updates will be posted at this same URL.
